## 앱 소개
하루 중 랜덤한 시간에 알림을 보내 그 순간의 사진을 기록하는 라이프로그 앱입니다.
꾸미지 않은 자연스러운 일상을 기록할 수 있습니다.

## 주요 기능
- 랜덤 알림으로 하루 한 번 사진 기록
- 캘린더에서 날짜별 사진 확인
- 기록 완료 알림

## 개발 환경
- xcode
- Swift
- UIKit
- Storyboard
- UserNotifications
